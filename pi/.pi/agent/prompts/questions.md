---
description: Identify decisions that require my input
---

Review the current task and changes for decisions that require my input.

- Ask only questions that need a judgment, preference, or product decision from me.
- Do not ask questions you can answer by inspecting the code or documentation.
- For each question, briefly explain why the decision matters.
- Present the available options and their practical trade-offs.
- Recommend an option when you have enough context.
- Group related questions and keep them concise.
- Ask questions one at a time. After I answer, automatically continue with the next question.
- If no decisions require my input, say so clearly.
